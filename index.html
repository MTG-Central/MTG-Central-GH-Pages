MTG Central

The future of local MTG trading

Our Project

Magic The Gathering Central is a site designed to facilitate trading between local players.

The goal of MTGC is to be a user friendly site that allows players to trade with a larger network, while still avoiding the hassle of online trading.

Project Architecture

  • Firebase
  • Angular
  • HTML
  • Sass
  • UI-Router
  • Bootstrap

User Stories

  • Anyone can sign in using Facebook so that they can create a listing (8)
  • Anyone can view other listings so that they can see others' have / wants
  • Anyone can search for cards so that they can find only posts with that card
  • An authenticated user can create a listing so that they can display the cards they have/want
  • An authenticated user can search for a card so that they can add it to their have/want list
  • An authenticated user can view results of their search so that they can add cards to their have/want list
  • An authenticated user can remove a card while creating their list so that they can fix mistakes
  • An authenticated user can edit their listing so that they can update card quantity and delete cards
  • An authenticated user can link to a user's Facebook so that they can contact them via Facebook

Stretch Goals

  • Integrate with Google maps
  • Category for shop posts / advertisement
  • Price guide
  • Card images display on hover
  • Users can chat on the site
  • Users can track their trade history
  • Can add feedback for users
